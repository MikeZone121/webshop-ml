## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MikeZone121/webshop-ml
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `.env`
   ```sh
    VITE_API_URL=https://dummyjson.com/;
   ```
4. Start project
   ```sh
   npm run dev
   ```
